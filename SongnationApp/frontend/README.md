# Songnation (Simplified)

Este repositório foi simplificado para servir como uma tela inicial (landing) estática
para desenvolvimento. A interface atual é uma página única que mostra o nome
"Songnation" centralizado e com uma animação de rotação.

Como executar localmente

1. Abra um terminal na pasta `SongnationApp`.
2. Execute:

```bash
# usa npx para servir a pasta localmente na porta 8080
npx serve . -l 8080
```

Abra http://localhost:8080 no navegador.

Observações

- Quasar-specific files and dependencies were removed or neutralized so this
	folder behaves as a plain static site. If you want to restore the Quasar
	scaffolding, re-add the original dependencies and configs.
